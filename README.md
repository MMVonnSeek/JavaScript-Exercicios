# Exercícios e Exemplos de JavaScript  
### Material Oficial: Professor Max Muller – Turma de Desenvolvedor Front-End

Este repositório reúne alguns dos exercícios, exemplos, desafios e materiais de JavaScript que **utilizei como professor** durante as aulas da turma de **Desenvolvedor Front-End**.  
Aqui está concentrado o conteúdo didático trabalhado em sala, organizado para facilitar a revisão e prática dos alunos.

---

## Sobre este material

Todo o conteúdo deste repositório foi preparado e utilizado por **mim, professor da turma**, ao longo das aulas.  
Ele serve tanto como:

- Base de estudos para os alunos
- Registro das atividades aplicadas em sala
- Repositório de exemplos comentados e resolvidos
- Espaço para desafios complementares e prática guiada

---

## Conteúdos Incluídos

-  Exemplos práticos apresentados em aula  
-  Atividades sobre funções, arrays, objetos e manipulação de dados  
-  Estruturas de repetição e condicionais  
-  Manipulação do DOM  
-  Mini projetos usados para demonstração  
-  Eventos, formulários e interatividade  
-  Desafios extras propostos aos alunos  

---


## Objetivo Pedagógico

Como professor, minha intenção com este repositório é:

- Facilitar a revisão para provas e avaliações  
- Disponibilizar exemplos reais utilizados em sala  
- Criar um ambiente para que os alunos pratiquem e evoluam  
- Manter um registro acessível das atividades da turma  

---

## Pré-requisitos

- Navegador atualizado  
- Editor de código (VS Code recomendado)  
- Noções básicas de HTML e CSS  
- Motivação para aprender e praticar
